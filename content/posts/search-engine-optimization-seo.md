# Search Engine Optimization

Search Engine Optimization atau biasa disebut SEO, merupakan sebuah metode untuk melakukan optimasi pada website supaya dapat muncul dipencarian Search Engine (Contoh google.com).

Cara nya adalah melakukan optimasi isi konten pada website sesuai dengan **keyword** yang di inginkan.

## Cara Kerja (Ambil case menggunakan Google.com)

1.  **Crawling**: Googlebot mencari halaman baru melalui link dari website satu ke website lain
2.  **Indexing**: Google melakukan analisis konten (teks, gambar, struktur) dan menyimpanya ke database
3.  **Ranking**: Saat user mencari keyword Google akan memilih konten paling relevan berdasarkan (**E-E-A-T**) untuk ditampilkan ke posisi teratas

## Strategi untuk terindeks oleh google

-   **Google Search Console**: Untuk request Indexing
-   **Sitemap.xml**: file intruksi yang perlu disiapkan diwebsite sebagai pedoman saat bot google melakukan indexing memahami struktur websitenya
-   **Robots.txt**: file untuk menentukan halaman yang tidak boleh dan boleh dijelajahi

Untuk SEO ini juga dapat ditambahkan untuk optimize pencarian by **AI Chat** (Gemini, ChatGPT etc) dengan memanfaatkan **JSON-LD** untuk memberitahu isi website ini tentang apa dan digunakan AI untuk menentukan relevansinya dengan yang chat request dari user mereka.
