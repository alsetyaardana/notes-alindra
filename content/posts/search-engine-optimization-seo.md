# Panduan Search Engine Optimization (SEO)

*Catatan lengkap oleh Alindra Setya Ardana tentang Search Engine Optimization dan strategi untuk mendapat ranking terbaik di Google.*

## Apa itu SEO?

Search Engine Optimization atau biasa disebut SEO, merupakan sebuah metode untuk melakukan optimasi pada website supaya dapat muncul di pencarian Search Engine (contoh: google.com).

Caranya adalah melakukan optimasi isi konten pada website sesuai dengan **keyword** yang diinginkan.

## Cara Kerja Google (Studi Kasus)

### 1. Crawling
**Googlebot** mencari halaman baru melalui link dari website satu ke website lain

### 2. Indexing  
Google melakukan analisis konten (teks, gambar, struktur) dan menyimpannya ke database

### 3. Ranking
Saat user mencari keyword, Google akan memilih konten paling relevan berdasarkan **E-E-A-T** (**Experience, Expertise, Authoritativeness, dan Trustworthiness**) untuk ditampilkan ke posisi teratas

## Strategi Agar Terindeks Google

### Tools Utama:
- **Google Search Console**: Untuk request indexing manual
- **Sitemap.xml**: File instruksi yang perlu disiapkan di website sebagai pedoman saat bot Google melakukan indexing dan memahami struktur website
- **Robots.txt**: File untuk menentukan halaman yang tidak boleh dan boleh dijelajahi

## SEO untuk Era AI Modern

Untuk SEO ini juga dapat ditambahkan untuk optimize pencarian by **AI Chat** (Gemini, ChatGPT etc) dengan memanfaatkan **JSON-LD** untuk memberitahu isi website tentang apa dan digunakan AI untuk menentukan relevansinya dengan chat request dari user mereka.

### Keunggulan JSON-LD:
- Membantu AI memahami context website
- Meningkatkan peluang muncul di AI search results
- Struktur data yang lebih terorganisir
