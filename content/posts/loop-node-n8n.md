# Tutorial Loop Node N8N

*Catatan teknis oleh Alindra Setya Ardana tentang penggunaan Loop Node yang merupakan salah satu fitur paling powerful di N8N.*

Node Loop di N8N merupakan salah satu yang powerful, untuk stopper menunggu semua node berjalan selesai terlebih dahulu sebelum melanjutkan ke node berikutnya.

## Cara Kerja Loop Node

Loop Node berfungsi sebagai kontrol flow yang memastikan semua iterasi dalam loop selesai dieksekusi sebelum workflow melanjutkan ke tahap berikutnya. Ini sangat berguna untuk:

- **Batch Processing**: Memproses data dalam jumlah besar secara bertahap
- **Rate Limiting**: Mengontrol kecepatan eksekusi untuk menghindari API limit
- **Sequential Operations**: Memastikan operasi berjalan secara berurutan
